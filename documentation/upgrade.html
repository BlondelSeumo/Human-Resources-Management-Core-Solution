Upgrade Guide

Auto update
  • Backup all files and database before starting the installation of updates (including language files if you are using custom_lang.php file to overwrite translation text) and review the changelog.
  • Before starting UPDATE process, please increase the max_execution_time (Recommendation 2000 sec) and upload_max_filesize (Recommendation 128MB) on your hosted server.
  • You must install the previous versions to update the new version.
  1. Go to "Settings" > "App Settings" > "Update" Now, you can see the current version of the application as well as the new versions (if any new upgradable version is available).

  2. Now, you can see the current version of the application as well as the new versions (if any new upgradable version is available).

  3. Just click on the new version Update button on your application and it takes sometimes to complete the update. .

Manual Update:
  • Before starting the installation of updates, please increase the max_execution_time (Recommendation 2000 sec) and upload_max_filesize (Recommendation 128MB) on your hosted server.
  • Backup all files and database before starting the installation of updates (including language files if you are using custom_lang.php file to overwrite translation text) and review the changelog.
  • You must install the previous versions to update the new version.
  • If the auto update failed, you will get the manual update option below, Download the update file zip from Step 1 and upload the zip in Step 2 (Don't rename the zip, otherwise this update will fail.)
  1. Go to "Settings" > "App Settings" > "Update" Now, you can see the new versions (if any new upgradable version is available).

  2. Step 1, You just click the download update file button and save the zip file on your desktop

  3. Step 2, Browse the download update zip and click on Update button and it takes sometimes to complete the update.
