Version 1.7.1 – 12 January 2023
[Updated] Maintenance update
Alert: v1.7.1 is a prerequisite of v1.8. For that you have to enable manual update and set PHP version ">= 8.0" on the hosted server.
  • To enable manual update, go to the following file path on your hosted server: src/config/Installer.php
  • Open the Installer.php file and change the 49 no line 'manual_update' => false to 'manual_update' => true

Please follow the video for more clearance.
