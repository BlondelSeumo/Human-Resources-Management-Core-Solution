Installation in Local Environment

Please follow the procedure.

  1. Make sure you have met environment requirements.
    • PHP Version == 8.0 or 8.1
    • MySql Version >= 5.6+
    • Maria DB Version >= 10.2+
  2. Now, download the zipped file from CodeCanyon.
  3. Unzip the zipped file. Inside the unzipped folder you will get two folders "Documentation" and "upload"
  4. Move all the items from the "upload" folder to local directory from where you want to run the application
  5. Open the folder in the Editor (phpstorm, vscode etc.)

  6. Go to src folder and open .env file: update APP_INSTALLED=true

  7. Update the database credentials in the .env file.

    DB_CONNECTION=mysql
    DB_HOST=127.0.0.1/localhost
    DB_PORT=3306
    DB_DATABASE=your_database_name
    DB_USERNAME=your_database_user_name
    DB_PASSWORD=your_database_password

  8. Now navigate to src folder in command line and run the below commands
    • php artisan optimize:clear
    • php artisan migrate:fresh --seed
    • php artisan storage:link

  9. In this src folder open another terminal and run the below command in order to complete all the queued jobs generated in the application
    php artisan schedule:work

  10. Now go back to the root directory of your application and run the below command on a terminal
    php -S localhost:8000 -t ./

  11. Then you will get a URL in terminal: ex (http://127.0.0.1:8000)

  12. Copy and paste the url on a browser.

  13. Now, complete the installation process. And your application should be ready to use locally.
